# Decisión: filter-strategy

**Decisión:** Filter pasivo: vista detalle ya separa por estado. Sin UI de filter extra v0.1.

## Alternativas
- Multi-filter por rol × estado × prioridad — overkill para 4 roles.
- Segmented control Pills — innecesario para 3 estados.

## Justificación
Vista detalle ya es un filter implícito (por rol). 3 listas visibles cubren los 3 estados.
