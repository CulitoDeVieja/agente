# Decisión: ux-microinteracciones

**Decisión:** Minimal v0.1: hover feedback (color + border) y press state (scale 0.98). Sin animaciones narrativas.

## Alternativas
- Framer Motion — 30KB, innecesario para hover.
- Zero feedback — UX pobre, user no sabe qué es clickeable.
- Ripple effect (Material) — estilo chocante con paleta oscura minimal.

## Justificación
Hover/press con Tailwind alcanza. Microinteracciones "delightful" no agregan valor en un dashboard de trabajo.
