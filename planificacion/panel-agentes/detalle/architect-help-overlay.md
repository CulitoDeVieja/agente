# Decisión: help-overlay

**Decisión:** Atajo `?` (shift+/) muestra overlay con lista de shortcuts. Cierra con Esc o click.

## Alternativas
- Tutorial interactivo — overkill.
- Tooltips individuales — ya cubiertos.
- Documentación externa — fricción.

## Justificación
Overlay simple con tabla shortcuts. 10 líneas de JSX.
