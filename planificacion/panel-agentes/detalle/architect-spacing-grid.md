# Decisión: spacing-grid

**Decisión:** Spacing scale 4-based: 4/8/12/16/24/32/48 px (Tailwind 1/2/3/4/6/8/12). Grid 12-col cuando aplique (dashboard). Gap estándar entre cards: 16px.

## Alternativas
- Scale 8-based — menos fino-grained.
- Grid 16-col — overkill para 4 cards.
- Sin sistema (valores arbitrarios) — incoherencia visual.

## Justificación
Tailwind default (4px base) es estándar y cubre el caso.
