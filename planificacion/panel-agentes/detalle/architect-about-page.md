# Decisión: about-page

**Decisión:** Pequeño footer en config modal: "panel-agentes v0.1.0 — <link github>". Sin About page separada.

## Alternativas
- About page completa con licencias — v0.2.
- Link directo a README del repo — válido.
- Sin info de versión — user no sabe qué corre.

## Justificación
Mínimo requerido: versión + link. En footer del modal config.
