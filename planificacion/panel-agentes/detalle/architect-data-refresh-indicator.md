# Decisión: data-refresh-indicator

**Decisión:** Texto tenue en Header: "actualizado hace Xm" + botón ↻ al lado. Durante refresh, spinner reemplaza ícono del botón.

## Alternativas
- Progress bar global — intrusivo.
- Solo botón sin info — user no sabe si la data es stale.

## Justificación
Info + acción juntos = low-friction verificar estado.
