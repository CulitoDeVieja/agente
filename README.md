# Agentes

Manifiestos de comportamiento, skills y contexto de cada agente del sistema.

Cada archivo en `agentes/` define:
- **Rol** — qué es el agente.
- **Funciones** — qué hace.
- **Skills** — qué habilidades aplica.
- **Comportamiento** — reglas de interacción.
- **Contexto** — qué necesita para operar.
- **Firma** — cómo se identifica en commits.

Una IA que tome uno de estos alias debe leer el manifiesto primero.
